Hexo 图片插入细则

HTML 标签方式（推荐用于缩放）

关键要点

  1. 路径格式： 使用 /文件夹名/图片名.jpg（绝对路径，以 / 开头）
  2. 尺寸控制： width 设置宽度，height: auto 保持比例
  3. 中文路径： 支持中文文件夹名，无需URL编码
  4. 常用尺寸： 150px-300px 适合博客文章
